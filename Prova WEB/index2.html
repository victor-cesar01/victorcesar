Curso Programador WEB

Sobre o curso de Programador WEB

Você aprenderá a planejar, programar e publicar websitescom métodos que permitem a inserção e manipulação de conteúdo dinâmico, além de desenvolver interfaces de acordo com os padrões web.

HTML

O HTML é uma linguagem base para desenvolver qualquer site. É uma linguagem baseada em marcação, onde marcamos os elementos para definir quais informações a página vai subir. A linguagem HTML foi criada na década 1990 e suas especificações são controladas pela W3C (World Wide Web Consortium).

CSS

O CSS é uma linguagem de folha de estilo composta pr ''camadas'', criado com propósito de estilizar as páginas HTML, ou seja, definiar a aparência das páginas, para deixar visualmente mais bonitas e agradáveis, podendo alterar a fonte, cor, posicionamento dos elementos, layout e muito mais. Uma das maoires vantagens de se utilizar o CSS é que ele efetua a separação entre o formato da página e o contúdo de um documento, proporcionando assim uma maior flexibilidade e controle de como as características serão exibidas, além de reduzir a duplicação no conteúdo estrutural de uma página.
